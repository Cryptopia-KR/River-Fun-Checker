Privacy Policy for River Checker
Last Updated: October 30, 2025

---

Overview
River Checker is a Chrome extension that helps users check Twitter account rankings and statistics on the River platform (formerly Satoshi Protocol). We are committed to protecting your privacy.

---

Data Collection
River Checker collects and stores the following data:

- Last searched username: Stored locally in your browser using Chrome's storage API for convenience. This data never leaves your device.

We do NOT collect:

- Personal information to our servers
- Browsing history
- Analytics or usage statistics
- Any data for sale to third parties

---

Third-Party Services
This extension communicates with:

- River API (api-v2.satoshiprotocol.org)
  Purpose: Fetch public Twitter account statistics and rankings
  Data Sent: Twitter usernames (public information)
  Data Received: Public ranking information, rewards, scores
  Privacy Policy: https://app.river.inc

- Unavatar.io
  Purpose: Display user profile images
  Data Sent: Profile image URLs from River API
  Privacy Policy: https://unavatar.io

- Twitter/X Integration
  The extension adds a lookup button on Twitter/X profile pages
  No data is sent to Twitter/X by this extension
  Only publicly available profile information is accessed

---

Data Security

- All data stored by this extension remains on your local device
- Chrome's secure storage API is used
- No data is transmitted to any servers we control
- All API requests are made over HTTPS

---

Permissions
This extension requires:

- activeTab: To add the River lookup button on Twitter/X profile pages
- storage: To save your last searched username locally
- host_permissions: To access River API, Twitter/X pages, and profile images
